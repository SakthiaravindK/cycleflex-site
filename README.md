# CycleFlex Website (Static)

## Edit products + WhatsApp number
Open `app.js` and change:
- `supportWhatsApp` to your WhatsApp number (countrycode + number, no +)
- edit products list (title, price, sizes, etc.)

## Run locally
Just open `index.html` in your browser.
